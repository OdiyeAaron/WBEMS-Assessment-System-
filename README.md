WBEMS-Assessment-System
An AI-powered assessment system (WBEMS) that utilizes machine learning to track student competence and engagement through real-time behavioral data.
 Overview
The Web-Based Evaluation and Management System (WBEMS) is designed to modernize how student progress is monitored. By leveraging machine learning, the system moves beyond traditional testing to analyze real-time behavioral data, providing a more holistic and data-driven view of student engagement and academic competence.
 Tech Stack
  Language: Python
  Machine Learning: Scikit-learn
  Backend: Flask / Python
  Database: MySQL
  Version Control: Git
 Project Structure
  train_model.py: The machine learning pipeline used to train the assessment model.
  requirements.txt: Contains all necessary Python packages and dependencies.
  python/: Directory containing core application logic and scripts.
 Installation & Setup
  Clone the repository:
   git clone https://github.com/OdiyeAaron/WBEMS-Assessment-System.git

  Navigate to the project directory:
   cd WBEMS-Assessment-System

  Install dependencies:
   pip install -r requirements.txt

  Train the ML Model:
   python train_model.py

   Run the Application:
   python app.py

 Key Features
Behavioral Analytics: Uses ML to interpret student engagement levels based on interaction data.
  Competence Tracking: Dynamically assesses mastery of subjects rather than relying on static grades.
  Automated Insights: Generates real-time feedback for educators to identify students who may need additional support.
